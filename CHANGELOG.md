# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-06-21

### Added

- **Workspaces & Git versioning** — upload HTML assets, in-app preview, version history, diff comparison, and restore
- **Review Mode** — click elements on HTML preview to annotate changes; copy structured AI modification prompt
- **Share links** — read-only preview or source collaboration with password, expiry, and access limits
- **MCP integration** — HTTP Streamable MCP at `/api/mcp` for Cursor, Claude Code, and other AI tools
- **Personal Access Tokens** — `adh_`-prefixed tokens with READ / READ_WRITE scopes
- **Global search** — find workspaces, repositories, and files across the platform
- **Recycle bin** — soft-delete workspaces and repos with restore within retention period
- **Audit logs** — system admin audit trail for key operations
- **Internationalization** — Chinese and English UI
- **In-app help** — feature catalog, interactive guides (Review Mode), and changelog drawer
